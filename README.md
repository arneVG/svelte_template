# svelte app

This is a modified project template for [Svelte](https://svelte.dev) apps.
This version adds prettier and eslint with configs and removes some code that I personally never use.
Find the unmodified version at https://github.com/sveltejs/template.

```bash
npx degit arneVG/svelte-template svelte-app
cd svelte-app
npm install
npm run dev
```
